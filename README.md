# Prompt Engineering Portfolio

## Objective
Demonstrate how structured prompts improve AI output quality.

## Example

### Basic Prompt
"Explain climate change."

### Optimized Prompt
"Explain climate change in simple terms for beginners. Include causes, effects, and real-world examples in under 150 words."

## Improvements
- Clear constraints  
- Defined audience  
- Output control  
- Better structure  

## Result
Higher clarity and consistency in AI responses.
